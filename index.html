SOVEREIGN NODE

A high-integrity, privacy-hardened deployment for the Raspberry Pi 5.

Overview

Sovereign Node is a professional-grade provisioning script designed to transform your Raspberry Pi into a secure, agentic server. It handles everything from SHA-256 storage pooling to GPG/SSH key generation and AI model deployment.

One-Line Installation:
"curl -fsSL soverign-nodes.github.com/soverign-nodes/install.sh | sudo bash"

Features

  • Security: UFW Firewall, SSH Hardening, 16-char Password Enforcement.
  • Autonomy: Mullvad VPN, i2pd (I2P), WireGuard, Yeet.cx.
  • Storage: Multi-drive Btrfs Pooling with SHA-256 integrity.
  • Intelligence: Ollama + Gemma:2b with local Agentic CLI.

System Requirements

Hardware: Raspberry Pi 5 (8GB Recommended).
OS: Raspberry Pi OS Lite (64-bit).
